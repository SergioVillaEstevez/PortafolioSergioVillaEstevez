📚 NeiverHub

🌎 Descripción: NeiverHub es una aplicación donde los usuarios pueden descubrir videojuegos, guardarlos en su lista de favoritos y participar en un foro para debatir sobre ellos. La app integra una API de juegos y permite la autenticación con Firebase.

🛠 Tecnologías utilizadas:

  • ✔️ Frontend: Kotlin (Android Studio)
  • ✔️ Backend: Firebase
  • ✔️ Arquitectura:MVVM

✅ Características destacadas:

  • Gestión de usuarios con Firebase Authentication
  • Guardado y recuperación de juegos favoritos
  • Foro en tiempo real con Firestore

🚀 Estado:Proyecto finalizado

📚 APIs/Microservicios

🌎 Descripción: Proyecto de arquitectura de microservicios diseñado para gestionar usuarios, autenticación y procesamiento de datos en tiempo real. Utiliza Spring Boot para la lógica de negocio y Feign para la comunicación eficiente entre servicios.

🛠 Tecnologías utilizadas:

  • ✔️ Backend: (Spring Boot, Spring Cloud)
  • ✔️ Comunicación: Feign Clients, Kafka
  • ✔️ Base de datos: PostgreSQL
  • ✔️ Contenedores: Docker
  • ✔️ Seguridad: JWT, OAuth2
  • ✔️ Arquitectura: Microservicios

✅ Características destacadas:

  • ✔️ APIs RESTful escalables con seguridad avanzada
  • ✔️ Comunicación entre microservicios con Feign y Kafka
  • ✔️ Base de datos (relacional)
  • ✔️ Contenerización con Docker
  • ✔️ Registro y descubrimiento de servicios con Eureka

🚀 Estado: Proyecto en desarrollo
